Bem Vindo ao sistema de gerenciamento de Biblioteca!

Esse projeto consta com funções de cadastro de livros, autores e emprestimos. Além de lidar com devoluções, atualizações de dados, disposição das informações e geração de estatísticas com base no contexto da escola em que estudei.

As tecnologias usadads neste projeto foram HTML, CSS, Java Scrip, PHP (principalmente) e Banco de Dados MySQL
